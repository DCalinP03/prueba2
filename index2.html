Hosting Gratuito

Normalmente ofrecen recursos muy limitados, suelen icluir publicidad con servicios de pago para dicho hosting.

Ventajas: Aprender y probar sin ningun gasto economico

Desventajas: Recursos muy reducidos y baja calidad.

Hosting Compartido

Los servidores suelen ser maquinas con prestaciones muy altas. La mayor parte de los sitios web no necesitan mucho poder de computo. Un mismo servidor se utiliza para alojar varias páginas web.

Ventajas: Bastante economico, estan muy bien equipados.

Desventajas: Si alguna pagina web consume muchos recursos afecta a todas las paginas web.

Hosting para Revendedores

Funciona igual que un Hosting Compartido, con la unica diferencia de que hay una menor cantidad de cuentas individuales.

Ventajas: Todas las licencias estan incluidas en el servicio

Desventajas: Cuando hay fallo en el sistema se debe de esperar al proveedor solucione el fallo.

Hosting VPS

Se trata de un servidor físico real, y mediante la tecnologia de virtualización, en el se crean varias máquinas virtuales

Ventajas: No dependemos de otros sitios web, podemos mejorar el plan personalmente.

Desventajas: Son mas caros, hay que tener mas conocimientos para administrarlo.

Hosting Dedicado

Un servidor comprado al completo para nosotros.

Ventajas: Todos los recursos son para nosotros, libertad para administrar y configurar dicho servidor.

Desventajas: Su excesivo coste, debemos de tener buenos conocimientos para mantenerlo, administrarlo y configurarlo.

Hosting en la Nube o Cloud Hosting

Se trata de un avance respecto a los servidores tradicionales, es un servidor en la nube, el trabajo se reparte entre varias máquinas físicas

Ventajas: No hay caidas o interferencias en el sistema, buena adaptación y buena relación calidad/precio

Desventajas: No garantiza una total privacidad de los datos sensibles.
